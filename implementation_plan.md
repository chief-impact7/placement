# 고등부 성적표 개발 및 시스템 통합 계획 (v23)

## 1. 개요
- 고등부 학생들을 위한 전문적인 성적 분석표를 구현하고, 학부별(초/중/고) 테마 시스템을 완성합니다.

## 2. 작업 상세
### 2.1 ReportCard.jsx (v2)
- **3-Theme System**: 
    - 초등부: Orange (활기찬 느낌)
    - 중등부: Indigo (신뢰감 있는 느낌)
    - 고등부: Slate (아카데믹하고 전문적인 느낌)
- **데이터 구조**:
    - 고등부 필수 6개 영역 + SUM 필드 매핑.
    - `(30%)`, `(av)` 접미사를 활용하여 상위권 및 평균 데이터 추출.
    - 계산 로직 없이 시트 데이터를 그대로 사용하도록 보강.
- **레이아웃**:
    - 중앙: 3개 행(내 점수, 30%, 평균)으로 구성된 요약 테이블.
    - 하단: 좌측(Bar Chart), 우측(Radar Chart) 2단 구성.

### 2.2 App.jsx 통합
- `showReport` 상태 변경 시 `spreadsheetId` 등 필수 프로프 전달 보강.

### 2.3 빌드 및 배포
- `npm run build` 및 `firebase deploy` 완료.

## 3. 검증 계획
- 고등부 학생 선택 후 성적표 출력 시 Slate 테마가 적용되는지 확인.
- 테이블에 내 점수, 상위 30%, 평균 데이터가 정확히 3행으로 표시되는지 확인.
- 하단 차트 2개가 좌우로 정상 배치되는지 확인.
