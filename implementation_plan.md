# 선택 학생 구글 시트 내보내기 구현 계획 (v18)

## 1. 개요
- 학생 목록에서 체크된 학생들을 별도의 구글 시트 파일로 생성하여 구글 드라이브에 저장하는 기능을 추가합니다.

## 2. 작업 상세
### 2.1 sheetsService.js
- `exportToNewSpreadsheet(title, headers, rows)`: 신규 스프레드시트를 생성하고 데이터를 삽입하는 핵심 로직 추가.

### 2.2 App.jsx
- `handleExportSelected`: 선택된 학생들을 필터링하고 모든 성적 필드를 수집하여 시트 데이터를 구성하는 로직 구현.
- **UI 추가**: 학생 목록 상단 검색바 옆에 선택된 인원수를 표시하는 '시트로 내보내기' 버튼 추가 (조건부 렌더링).

### 2.3 빌드 및 배포
- `npm run build` 실행.
- `firebase deploy --only hosting` 실행.

## 3. 검증 계획
- 학생 선택 시 버튼이 나타나는지 확인.
- 버튼 클릭 시 구글 드라이브에 파일이 생성되고 링크가 열리는지 확인.
